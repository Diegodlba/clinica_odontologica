Examen Final Práctico - Clínica Odontológica
Alumno Diego de la Barrera

Bienvenido/a a mi página web de la Clínica Odontológica. ¡Estás presenciando un examen final práctico de la materia de Backend!

En este proyecto, he creado una aplicación para una clínica odontológica, abordando los siguientes puntos:

  • - Clase de Negocio y Base de Datos en H2
  • - ORM (Object-Relational Mapping)
  • - Utilización de Repository
  • - Utilización de Service
  • - Utilización de Controllers
  • - Testeo con JUnit
  • - Captura Global de Excepciones
  • - Uso de Log4J
  • - Front End: Invocaciones JavaScript con Fetch

Puedes explorar todas las funcionalidades de la aplicación navegando por las diferentes secciones en la barra de navegación.
